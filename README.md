## Image Cloud

> Примечание: проект был создан для изучения инструментов Multer и FS в контексте Node.js фреймворка NestJS

Проект представляет из себя **облачное хранилище**, предоставляющее возможность загрузки и удаления, а также встраивания и просмотра изображний по ссылке
